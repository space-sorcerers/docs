## Filters
<div style={{display: "flex", justifyContent: "space-between", alignItems: "center"}}>
<div>
<h3>with &lt;component type&gt;</h3>
</div>
<div style={{textAlign: "right"}}>

`IEnumerable<EntityUid> -> IEnumerable<EntityUid>`
</div>
</div>
Фильтрует входные данные, оставляя сущности с указанным компонентом.
Эту команду можно инвертировать с помощью `not`.

<div style={{display: "flex", justifyContent: "space-between", alignItems: "center"}}>
<div>
<h3>prototyped &lt;prototype&gt;</h3>
</div>
<div style={{textAlign: "right"}}>

`IEnumerable<EntityUid> -> IEnumerable<EntityUid>`
</div>
</div>
Фильтрует входные данные, оставляя сущности, созданные из указанного прототипа.
Эту команду можно инвертировать с помощью `not`.

<div style={{display: "flex", justifyContent: "space-between", alignItems: "center"}}>
<div>
<h3>paused</h3>
</div>
<div style={{textAlign: "right"}}>

`IEnumerable<EntityUid> -> IEnumerable<EntityUid>`
</div>
</div>
Фильтрует входные данные, оставляя сущности, которые в данный момент приостановлены.
Эту команду можно инвертировать с помощью `not`.

<div style={{display: "flex", justifyContent: "space-between", alignItems: "center"}}>
<div>
<h3>nearby &lt;range&gt;</h3>
</div>
<div style={{textAlign: "right"}}>

`IEnumerable?<EntityUid> -> IEnumerable<EntityUid>`
</div>
</div>
Фильтрует сущности, находящиеся рядом с входными, возвращая все сущности в пределах радиуса от них.

<div style={{display: "flex", justifyContent: "space-between", alignItems: "center"}}>
<div>
<h3>named &lt;name&gt;</h3>
</div>
<div style={{textAlign: "right"}}>

`IEnumerable<EntityUid> -> IEnumerable<EntityUid>`
</div>
</div>
Фильтрует входные данные, оставляя сущности с именем, соответствующим regex `$regex^`.
