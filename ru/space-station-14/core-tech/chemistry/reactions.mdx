---
title: "Reactions"
---


Прототипы реакций определяют рецепты с соотношениями реагентов, необходимых для химической реакции. Ниже приведены значения конфигурации YAML для реакций:

`type`: Тип прототипа. Всегда должен быть `reaction`.
`id`: Уникальный id, используемый игрой для идентификации реакции. В `PasacalCase`
`reactants`: Список реагентов, необходимых для реакции. Каждый реагент указывает своё соотношение и является ли он катализатором. Смотрите пример ниже для подробностей.
`products`: Список реагентов, создаваемых в результате реакции. Они будут добавлены в контейнер раствора, в котором происходит реакция, если есть место.
`effects`: Список эффектов и их свойств. Каждый эффект соответствует классу C#, реализующему IReactionEffect. Смотрите пример ниже для подробностей.

```yaml
- type: reaction
  id: PotassiumExplosion
  reactants:
    Water:
      amount: 1
    Potassium:
      amount: 1
  effects: # Эффекты реакции
    - !type:ExplosionReactionEffect
      # Дальности при реакции 1 калий + 1 вода (единичная реакция)
      devastationRange: 0.05
      heavyImpactRange: 0.1
      lightImpactRange: 0.15
      flashRange: 0.2
      scaled: true # Масштабируется пропорционально количеству калия и воды
      maxScale: 30 # Сила взрыва перестаёт масштабироваться при 30 калий + 30 вода

```

Вот ещё один пример. В нём есть катализатор и некоторые химические продукты, но нет эффекта реакции. Обратите внимание, что реагенты являются катализаторами, только если это прямо указано, в противном случае они расходуются.

```yaml
- type: reaction
  id: PolytrinicAcid
  reactants:
    SulfuricAcid:
      amount: 1
    Chlorine:
      amount: 1
      catalyst: True # False, если не указано
    Potassium:
      amount: 1
  products:
    PolytrinicAcid: 3
```
