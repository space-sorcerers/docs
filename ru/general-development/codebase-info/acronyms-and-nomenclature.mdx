title: "Аббревиатуры и терминология"
---

| Сокращение              | Значение                                                                                                             |
|:------------------------ |:------------------------------------------------------------------------------------------------------------------- |
| SS14                     | Space Station 14. Переделка Space Station 13 (SS13).                                                                |
| BYOND                    | Игровой движок для SS13.                                                                                           |
| RobustToolbox, Engine    | Игровой движок SS14. (Аналог BYOND).                                                                                 |
| Content, Content Pack    | «Игра», работающая на RobustToolbox. (Аналог SS13).                                                                 |
| CVar                     | Convar/Console Variable. Настраиваемое значение, которое можно изменить в конфигурационных файлах или через консоль.         |
| .yml, YAML               | YAML Ain't Markup Language. Используется для определения prototypes.                                                              |
| .toml, TOML              | Tom's Obvious Minimal Language. Как YAML, но для конфигов.                                                           |
| ECS                      | Entity Component System                                                                                             |
| IoC                      | [Inversion of Control](../../robust-toolbox/ioc)                                                                 |
| .dmi, DMI                | Формат спрайтов BYOND/SS13. Конвертируется в RSI для нашего использования.                                                 |
| .rsi, RSI                | Robust Station Images. Формат изображений SS14 (фактически папка).                                              |
| PVS                      | Potentially Visible Set. Предотвращает отправку сервером клиентам сущностей, находящихся вне зоны видимости.                            |
| PJB, PJB3005             | Лид проекта. Какого хера моё имя здесь?                                                                  |
| VSC, VSCode              | Visual Studio Code. Не путать с VS.                                                                             |
| VS                       | Visual Studio Community 2017/19. Не путать с VSC. IDE для тех, кто не может получить лицензию Rider.             |
| Rider                    | [Кроссплатформенная IDE для C#.](https://www.jetbrains.com/rider/) (Можно получить бесплатно для некоммерческого использования).  |
| Watchdog                 | Watchdog сервера SS14. Используется для логирования, обновления и общего управления выделенным сервером.                           |
| Lidgren                  | Сетевая библиотека.                                                                                                 |
| Box2D                    | Основа (сильно модифицированной) физической системы SS14.                                                             |
| Avalonia                 | UI-фреймворк, используемый для лаунчера.                                                                                 |
| Postgres, SQLite         | Базы данных.                                                                                                          | 
| EntityUid                | Entity Unique Identifier                                                                                            |
